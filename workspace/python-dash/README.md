# Python Dash Example

This project runs a Dash web app inside Interactive Studio's preview pane.

## Run

1. Open `app.py`.
2. Click `Run`.
3. Wait for the `App ready:` console message.
4. Use `Stop` to terminate the Dash server.

## Notes

- The runtime injects `DASH_HOST` and `DASH_PORT` environment variables.
- If `8050` is busy, the app auto-selects the next available port.
