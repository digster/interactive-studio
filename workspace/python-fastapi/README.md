# Python FastAPI Example

This project runs a FastAPI web app inside Interactive Studio's preview pane.

## Run

1. Open `main.py`.
2. Click `Run`.
3. Wait for the `App ready:` console message.
4. Use `Stop` to terminate the FastAPI server.

## Notes

- The runtime injects `HOST` and `PORT` environment variables.
- If `8050` is busy, the app auto-selects the next available port.
