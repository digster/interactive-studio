# 2026-02-18 — Implement All Missing Features

## Summary

Implemented all 6 remaining features for Interactive Studio that were scaffolded but not wired up:

### Feature 1: Preview Pane (Live HTML Rendering)
- Added `previewRefreshKey` and `requestRefresh()` to executionStore
- Created `src/lib/previewBuilder.ts` — builds self-contained HTML by inlining `<link>` and `<script src>` assets from open editor tabs or disk via `readFile`. Skips external URLs. Appends timestamp comment to force re-render.
- Created `src/hooks/usePreview.ts` — watches editor tabs, active tab, active project, and previewRefreshKey. Debounces 300ms then calls `buildHtmlPreview()` for HTML or sets raw content for markdown/svg/json/mermaid.
- Updated `PreviewPane.tsx` — renders `<iframe srcdoc sandbox="allow-scripts">` for HTML, `dangerouslySetInnerHTML` for SVG, `<pre>` for markdown/json/mermaid. Refresh button wired to `requestRefresh()`.
- Added `usePreview()` hook call in `App.tsx`

### Feature 2: Problems Tab
- Added `Problem` interface, `problems[]`, `addProblem()`, `clearProblems()` to executionStore
- Modified `setError()` to also push a problem entry when error is non-null
- Updated `ConsolePanel.tsx` — problems list with severity icons (AlertCircle/AlertTriangle), clickable file paths that open in editor, badge count on Problems tab label, clear button scoped to active tab

### Feature 3: Run Button & Code Execution
- Created `src/hooks/useCodeExecution.ts` — manages Python execution lifecycle via Tauri events. Saves file before running, clears console/problems, auto-opens console, listens for `python-output`/`python-exit` events. Parses Python tracebacks for file/line info. Web languages trigger preview refresh.
- Updated `EditorPane.tsx` — added run toolbar (h-8) with Play/Loader2 button, visible for executable languages. Shows ⌘+Enter shortcut hint.

### Feature 4: File CRUD UI (Context Menu)
- Created `src/components/ui/ContextMenu.tsx` — generic positioned dropdown via portal, boundary detection, outside click/Escape close
- Created `src/components/ui/ConfirmDialog.tsx` — delete confirmation modal with backdrop blur
- Created `src/components/ui/InlineInput.tsx` — auto-focused inline text input, pre-selects filename without extension, commits on Enter, cancels on Escape/blur
- Updated `FileTreeNode.tsx` — right-click context menu (Rename/Delete for files, New File/New Folder/Rename/Delete for folders), inline rename and new file/folder creation
- Updated `Sidebar.tsx` — FilePlus and FolderPlus icon buttons in Explorer header

### Feature 5: Settings Panel
- Created `src/components/settings/SettingsPanel.tsx` — modal dialog with all editor settings (Theme segmented control, Font Size +/- buttons with 10-24 range, Tab Size segmented control, Word Wrap/Minimap/Auto Save toggles, Auto Save Delay input)
- Updated `TopBar.tsx` — Settings button onClick opens SettingsPanel

### Feature 6: Project Creation
- Created `src/components/project/NewProjectDialog.tsx` — modal with name input (validation: non-empty, alphanumeric/hyphens/underscores, no duplicates), 2-column template grid (Blank, HTML, Python, Markdown, React, Mermaid). React/Mermaid use client-side template files written after blank project creation.
- Updated `TopBar.tsx` — "New Project..." button at bottom of project dropdown
